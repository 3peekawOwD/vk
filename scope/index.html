Внимание! Не убирайте право "offline", если планируете использовать токен с другого IP адреса (например на хостинге)

Инструкция

  1. Введите права и ID приложения (актуальный список прав смотрите на этой странице)
  2. Нажмите "готово"
  3. Затем нажмите "разрешить"
  4. Скопируйте часть адресной строки от access_token_{ID}=
